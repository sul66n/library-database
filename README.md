# Библиотечная система с бронированием книг

## Описание
База данных для учета книг, авторов, читателей, выдачи и бронирования.

## Возможности
- учет книг и авторов
- учет читателей
- выдача книг
- бронирование

## Структура репозитория
- schemas — схемы базы данных
- data — данные
- base_part — обязательная часть
- advanced_part — дополнительная часть
- tests — тесты
